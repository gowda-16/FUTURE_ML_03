# AI Resume Screening System

This project uses Machine Learning and NLP to automatically rank resumes based on job requirements.

## Technologies Used
- Python
- NLTK
- Scikit-learn
- TF-IDF
- Cosine Similarity

## Dataset
Resume Dataset

## Features
- Resume preprocessing
- Skill extraction
- Candidate ranking
- Missing skill detection

## How to Run

pip install pandas nltk scikit-learn

python resume_screening.py
